对不同部分的说明:

Content - 框的内容。
Padding - 内边距的区域，内边距是透明的。
Border - 边框。
Margin - 边界外的区域。外边距是透明的。

重要事项: 设置元素的 width 和 height 属性时，只需设置内容区域的宽度和高度。要计算元素的完整大小，还必须把内边距、边框和外边距加起来。
盒子
假如有这样一个div：
div {
  width: 320px;
  padding: 10px;
  border: 5px solid gray;
  margin: 0;
} 
元素总宽度 = 宽度 + 左内边距 + 右内边距 + 左边框 + 右边框 + 左外边距 + 右外边距
350px = 320px(宽度) + 20px（左+右内边距） + 10px（左+右边框） + 0px（左+右外边距）
